07

Admin Control Panel

User management · data control · logs · link manager · chatbot API

Admin

A restricted admin panel for managing all platform components. Control GEI and SNA datasets in PostgreSQL, manage registered users, review login logs, configure Groq API settings for the chatbot, manage Kumu/EDA external links (Kumu maps, Google Colab, Looker Studio), update research PDF links in Google Drive, and monitor chatbot conversation history.

User Management

View, edit, disable or delete registered user accounts. Manage roles and permissions.

GEI & SNA Data Management

Upload new GEI / SNA datasets directly to PostgreSQL, trigger data refresh, monitor data health.

Kumu & EDA Link Manager

Update external links for Kumu maps (Overall, Phnom Penh, Siem Reap, Battambang), Google Colab notebook, and Looker Studio dashboard.

Research & Report Manager

Add, update, or remove PDF report links from Google Drive. Manage all 7 publications in the reports library.

Chatbot API Configuration

Manage Groq API keys, switch between models, view chatbot conversation history and usage analytics.

Login Activity Logs

Track user login history, timestamps, and session activity. Export logs as CSV for auditing.
